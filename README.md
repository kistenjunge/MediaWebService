# MediaWebService

Simple node.js server that provides some REST resources to access media files stored on the file system.

The server uses [node.js](http://nodejs.org/) version v0.10.11.

The actual client code is provided under server/www. So far it's mainly [HTML5 Boilerplate](http://html5boilerplate.com)
and [video.js](http://www.videojs.com/) for video playback.

* Source: [https://github.com/kistenjunge/MediaWebService](https://github.com/h5bp/html5-boilerplate)


## Quick start

So far there's only one option:

1. Clone the git repo — `git clone
   https://github.com/kistenjunge/MediaWebService.git`
